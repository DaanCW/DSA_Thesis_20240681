# Comparison Analysis

Three-way cost comparison of Artefacts A, B, and C.

- `lifetime_comparison.ipynb` — the analysis notebook producing all figures
  and tables (crossover analysis, 10-year lifetime costs, sensitivity to L1
  gas price at 5/20/50 gwei, ETH at $1,720).
- `data-results/` — the measured gas results consumed by the notebook, one
  JSON per artefact.
- `SourceOfTruth_tables.xlsx` — source-of-truth tables backing the thesis
  tables.
- `fig1`–`fig6` — the figures as used in the thesis.
